AWS

AWS (Amazon Web Services) is a cloud platform provided by Amazon which offers a mixture of Platform as a Service(PAAS), Infrastructure as a Service(IAAS) and Software as a Service(SAAS) capabilities. I have experience in building projects which have utilised an AWS S3 and RDS services.

  • AWS S3: This is a service which provides object storage
  • AWS RDS: This is a service that provides hosted relational database capabilities.
