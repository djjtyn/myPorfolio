Amazon RDS

What is it?

Amazon RDS is an acronym for Amazon Relational Database Service. It is a service provided by Amazon Web Services (AWS). It allows a scalable relational database to be hosted in the cloud. The database can be queried as if it is stored locally.

Amazon S3

What is it?

Amazon S3 is an acronym for Amazon Simple Storage Service. It is a service provided by Amazon Web Services (AWS). It allows objects to be stored in scalable storage area's hosted in the cloud.
